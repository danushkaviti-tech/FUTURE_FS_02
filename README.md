# FUTURE_FS_02

## 📌 Description

FUTURE_FS_02 is a web application built using the Django framework. The project demonstrates the implementation of a backend-driven web application with database integration and dynamic web pages.

The application uses Django to manage server-side logic, handle requests, and interact with the database. HTML and CSS are used to design the user interface and create a clean and responsive layout.

This project showcases my ability to build full-stack web applications using Django, structure projects efficiently, and implement database-driven features.

## 🛠 Tech Stack

* Python
* Django
* HTML5
* CSS3
* SQLite (Django default database)

## 🚀 Features

* Django backend framework
* Database integration
* Dynamic templates
* Responsive UI with HTML & CSS
* Structured project architecture

## ▶️ Run Locally

1. Clone the repository

```
git clone https://github.com/danushkaviti-tech/FUTURE_FS_02.git
```

2. Navigate to the project folder

```
cd FUTURE_FS_02
```

3. Install dependencies

```
pip install -r requirements.txt
```

4. Run migrations

```
python manage.py migrate
```

5. Start the development server

```
python manage.py runserver
```

6. Open in browser

```
http://127.0.0.1:8000
```
